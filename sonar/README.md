# Sonar Mine Detector

声纳信号分类器 - 使用MLP神经网络区分岩石和水雷

## 项目简介

本项目是课程作业，使用UCI Sonar数据集，通过MLP神经网络区分水下目标是岩石（Rock）还是水雷（Mine）。最终模型测试准确率达到88.10%。

## 数据集

- 来源: UCI Sonar Dataset
- 样本数量: 208个
- 特征数量: 60个（60个频段的能量值）
- 类别: Rock（岩石）97个，Mine（水雷）111个

## 模型配置

| 参数 | 值 |
|------|-----|
| 网络结构 | (24, 12) |
| 正则化 alpha | 0.005 |
| 批次大小 | 8 |
| 学习率 | 0.001 |
| 早停耐心 | 10 |
| 随机种子 | 55 |

## 模型性能

- 训练准确率: 98.80%
- 测试准确率: 88.10%
- 过拟合: 10.70%

## 实验对比

### LASSO特征选择

| 配置 | 特征数 | 测试准确率 |
|------|--------|-----------|
| 不使用LASSO | 60 | 76.19% |
| 使用LASSO | 12 | 73.81% |

结论：不使用LASSO效果更好。

### 网络结构对比

| 网络结构 | 测试准确率 |
|----------|-----------|
| (24,12) + batch_size=8 | 85.71% |
| (32,16,8) | 83.33% |
| (24,12) 标准 | 76.19% |

## GUI功能

- 单个预测：手动输入60个特征值
- 批量预测：从CSV文件批量处理
- 视觉警告：检测到矿石时红色闪烁、窗口震动
